# Corner Shift Planter (Rust Plugin)

Corner Shift Planter allows players to plant seeds only in the four corner slots of a Large Planter Box while holding Shift.

## Features

- Per-player toggle (`/corner`)
- Permission controlled
- Configurable behavior
- Lightweight and optimized

## Permission

cornershiftplanter.use

Grant permission:
oxide.grant group default cornershiftplanter.use

## Command

/corner

## Installation

1. Place CornerShiftPlanter.cs in oxide/plugins/
2. Reload:
   oxide.reload CornerShiftPlanter
